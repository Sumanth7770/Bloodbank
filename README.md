# Bloodbank
.
The goal of this research was to create a blood management project system that would aid in the maintenance of blood donor records. Creating market strategies for blood donation, lobbying, and sensitization of blood donors becomes very difficult without quick and timely access to donor records. The blood management project system includes features that allow for quick access to donor records gathered across the country.
